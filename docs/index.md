# Fast TeuxDeux

FastAPI-based TeuxDeux clone application.

## Overview

A fast, modern task management application built with FastAPI.

## Features

- Task management
- Daily todo lists
- RESTful API

## Tech Stack

- Python
- FastAPI
- Modern frontend

## Getting Started

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the server: `uvicorn main:app --reload`
